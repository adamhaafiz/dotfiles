## Initial Setup

* [ ] Country -&gt; Netherlands
* [ ] Preferred Languages -&gt; English \(UK\)
* [ ] Input Sources -&gt; ABC - Extended

## Once Booted into Desktop

Download setup script:

```bash
curl -OL haafiz.io/mac
```

Review the script:

```bash
cat mac
```

If everything looks good, start your engines:

```bash
sh mac
```
